# Fintech AI System Design Guide for Product Managers

Interactive, diagrammatic learning guide for PMs transitioning to AI product/system design in fintech.

## Run

Open `index.html` in any browser.

## Customize

- Edit `script.js` -> `nodeData` to align with your specific fintech stack (cards, lending, wealth, payments).
- Edit `script.js` -> `patterns` to tune recommendation logic for risk appetite and regulator requirements.
- Update roadmap and quiz content in `index.html` for your market and compliance obligations.

## Included modules

- Fintech AI product stack mental model
- Interactive fintech decision lifecycle diagram
- Tradeoff simulator (latency, cost, quality, compliance risk)
- 30-60-90 PM transition roadmap for regulated launches
- Fintech-focused self-check quiz
